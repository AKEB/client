## Installation

The recommended way to install the library is through [Composer](http://getcomposer.org):

```sh
$ composer require tarantool/client
```


## Tests

To run unit tests:

```sh
$ phpunit --testsuite Unit
```

To run integration tests:
```sh
$ phpunit --testsuite Integration
```

To run all tests:
```sh
$ phpunit
```
